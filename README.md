# Converge

アーチェリーの着弾を記録し、戻って集まりを見る **判断補助アプリ** です。サイトを動かす方向は「目安」として出します。

**公開URL:** https://eita115115.github.io/archery-converge/

## 何を解決するか

- 的の前で6本の着弾をタップ記録
- 弓を置いた場所に戻り、矢の集まりとサイトを動かす方向の**参考**を確認
- 距離・風・サイト目盛り・履歴をスマホ1台で完結（PWA・オフライン対応）

## 誰向けか

- アーチェリー練習で「今日の集まり」と「サイトをどっちに動かすか」をメモしたい人
- 初心者向けのやさしい説明モード付き（デフォルトON）

## 使い方（3ステップ）

1. **準備** — 距離・風・サイトの数字を入力
2. **記録** — 的の前で着弾位置をタップ（6本）。長押しで微調整、「1本取り消し」で修正
3. **確認** — 戻って集まりとサイト提案の目安を見る

## 技術構成

| ファイル | 役割 |
|---------|------|
| `app.js` | UI・画面遷移・記録フロー |
| `geometry.js` | 的の座標・得点・マーカー |
| `physics.js` | 散布統計・サイト提案（参考値） |
| `beginner.js` | 初心者向け文言 |
| `sw.js` | オフラインキャッシュ |

検証: `npm run check`

## 計算の前提と注意（重要）

- サイト提案は**着弾の集まりから出した目安**であり、実弾道の保証ではありません
- 風・装備・フォーム・射場環境で結果は変わります
- **実際の調整はコーチ・安全確認・射場ルールを優先**してください
- 物理パラメータ（矢速・重量など）は任意入力。入れると目安の精度が上がる程度です
